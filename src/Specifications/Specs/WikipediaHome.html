Wikipedia Homepage Specification

Content

The homepage is the first page that the user see when he visits Wikipedia. From this page, the user should be able to easily search for interesting facts. The homepage is also the place where the user could get interesting topics of the day and various interesting historical facts.

Therefore the page should contain the following contents

Element Visibility
Logo visible
SearchBox visible
LoginLink visible
FeaturedArticle visible

Authentications and Authorisations

As an anonymous user when I attempt to view the page source, I shall get the page in readonly mode.
